🔱 BRAHMASTRA V11.5
The Omni-Penetration & Digital Forensic Suite

Brahmastra is an all-in-one cybersecurity toolkit designed for ethical hackers and forensic investigators. It integrates OSINT, vulnerability research, payload generation, and local delivery into a single, modular interface.

🔥 Key Features
🕵️ Digital Footprinting: Deep OSINT scanning using the Sherlock engine to map social media presence.

🛰️ IP Tracking: Instant Geo-IP location and ISP identification.

🛡️ Ethical Virus Builder: Cross-platform RAT (Remote Access Trojan) generator for authorized security audits.

🚀 Payload Delivery: Built-in HTTP server to host and deliver payloads within a local lab environment.

📝 Wordlist Generator: High-yield custom password list creator based on target profile data.

📱 IMEI Forensics: Luhn-validated mobile tracking and database lookup.

🔍 Google Dorks: Automated search strings for finding SQL vulnerabilities and admin panels.

🛠️ Installation & Setup
Prerequisites
Kali Linux (Recommended) or any Linux distribution.

Python 3.x

Root/Sudo Privileges

Setup Commands
Bash
# Clone the repository
git clone https://github.com/shouryacr7/BRAHMASTRA.git

# Navigate to the directory
cd BRAHMASTRA

# Install dependencies
sudo pip3 install -r requirements.txt

# Run the weapon
sudo python3 brahmastra.py
📂 Output Management
All generated payloads and custom wordlists are automatically saved to the brahmastra_loot/ directory. This folder is included in the .gitignore to prevent sensitive data from being uploaded to GitHub.
